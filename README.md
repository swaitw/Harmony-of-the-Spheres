# Gravity Simulator

Gravity Simulator, also known as Harmony of the Spheres, is an interactive Newtonian n-body gravity and space simulator that lets you simulate the solar system, n-body choreographies and exoplanetary systems. You can also create your own gravitational systems from scratch. 

## Online Version

There are two versions of gravity simulator available online, namely version one and two, where version two is the new version that has features that are not available in version one. 

### Version One

Version one of the simulator can be found at [gravitysimulator.org](https://gravitysimulator.org).

### Version Two

Version two of the simulator can be found at [gravitysimulator.org/version-2/scenarios/all/](https://gravitysimulator.org/version-2/scenarios/all/)

### Ads

To cover hosting and domain name costs, ads are displayed on the scenario menu pages, but not on the scenario pages where you run the simulations themselves. 

## Local Development

To run Gravity Simulator locally, simply run `npm start` from your terminal.
